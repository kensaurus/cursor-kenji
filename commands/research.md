# Firecrawl Research

## Purpose
Scrape web for current docs, latest patterns, and creative solutions for current task.

## Process

### 1. Define Scope
- Tech/problem to research
- Key search terms
- Success criteria

### 2. Scrape Sources (prioritize recent)
- Official docs (check version/date)
- GitHub repos (sort by recently updated)
- Stack Overflow (filter by votes + recent activity)
- Dev blogs from 2024-2025
- Release notes & changelogs

### 3. Validate Freshness
- Check publish/update dates
- Verify against current library versions
- Cross-reference multiple sources
- Discard outdated patterns (pre-2024 unless still valid)

### 4. Explore Creative Approaches
- Look beyond obvious solutions
- Check how other frameworks solve similar problems
- Find unconventional implementations
- Research emerging patterns & experimental features

### 5. Synthesize & Apply
- Compare approaches (pros/cons)
- Recommend best fit for current context
- Provide ready-to-adapt code
- Note gotchas and edge cases

## Output
- **Recommended approach** with rationale
- **Alternative options** if relevant
- **Code snippets** (current syntax, not deprecated)
- **Gotchas** to avoid
- **Source links** for reference
