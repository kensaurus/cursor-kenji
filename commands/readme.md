# /readme

> End-of-session doc sync — detect convention, smart change detection, stale-reference cleanup.

This command is a thin entry point. Two skills cover the README workflow:

- **`enhance-readme`** at `~/.cursor/skills/enhance-readme/SKILL.md` — visual showcase: hero image, feature tour grid, optional GIF, theme-aware screenshots via Playwright MCP.
- **`docs-writer`** at `~/.cursor/skills/docs-writer/SKILL.md` — text content, structure, accuracy.

Use `/readme` to explicitly trigger doc sync. Otherwise, say "update the README", "enhance README", or "make README prettier" and the appropriate skill auto-fires.
